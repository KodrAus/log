An arbitrary structured value.

This trait cannot be implemented manually

The Visit trait is always implemented for a fixed set of primitives:

  • Standard formats: Arguments
  • Primitives: bool, char
  • Unsigned integers: u8, u16, u32, u64, u128
  • Signed integers: i8, i16, i32, i64, i128
  • Strings: &str, String
  • Bytes: &[u8], Vec<u8>
  • Paths: Path, PathBuf
  • Option<T: Visit>
  • &'a T: Visit

Any other type that implements serde::Serialize + std::fmt::Debug will automatically implement Visit if the kv_serde feature is enabled.

Implementing Visit

Visit can be implemented for datastructures by implementing serde::Serialize and std::fmt::Debug.
